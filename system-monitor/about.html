What is the Argentquest Development Suite?

The Argentquest Development Suite is a comprehensive, self-hosted development and production environment built with open-source technologies. This containerized infrastructure provides enterprise-grade functionality while maintaining full control over your data and deployment.

Origins & AI Experimentation

This stack originated from the need to experiment with AI chat applications using OpenAI's GPT models and explore the rapidly evolving landscape of generative AI technologies. What began as a simple FastAPI backend for testing AI conversations has evolved into a comprehensive development suite that demonstrates practical alternatives to expensive cloud AI services.

The demo code showcases a FastAPI-based AI Storytelling Assistant with integrated world-building and RAG (Retrieval Augmented Generation) capabilities. Using OpenRouter as the LLM provider, the application demonstrates how to build production-ready AI applications without vendor lock-in to expensive cloud AI services. The codebase includes real-time WebSocket connections for AI-assisted writing, vector embeddings with PostgreSQL+pgvector, and background job processing for document ingestion.

Many of the AI storytelling concepts and architectural patterns demonstrated in this stack were inspired by Ink & Quill, a SaaS application for AI-assisted creative writing. This self-hosted implementation explores how such cloud-based AI writing tools can be rebuilt using open-source technologies and cost-effective infrastructure, demonstrating that sophisticated AI applications previously only available as expensive SaaS offerings can be developed and deployed independently.

OpenRouter Integration

The application uses OpenRouter to access multiple AI models (including GPT-5 Nano) through a single API, providing cost-effective AI capabilities with model flexibility. This approach eliminates the need for direct OpenAI API subscriptions while maintaining access to cutting-edge language models. The demo includes comprehensive cost tracking, token usage monitoring, and dynamic model configuration for optimal performance and cost management.

Container Architecture Overview
Frontend & Routing
Public-facing services and traffic management with SSL termination and subdomain routing.
  • Nginx Proxy Manager - Reverse proxy with Let's Encrypt SSL
  • Heimdall Dashboard - Centralized application launcher
  • System Monitor - Real-time container monitoring
Application Services
Core FastAPI applications providing REST APIs and WebSocket connections for AI-powered features.
  • FastAPI Production - Main application server
  • FastAPI Development - Hot-reload development server
  • Monitor API - Container control backend
Data Layer
Multi-database architecture supporting relational data, documents, vector search, and caching.
  • PostgreSQL + pgvector - Main database with vector search
  • MongoDB - NoSQL document database
  • Redis - In-memory cache and session store
  • MinIO - S3-compatible object storage
Database Management
Web-based administration interfaces for all database systems with full CRUD capabilities.
  • pgAdmin - PostgreSQL web administration
  • MongoDB Express - MongoDB web interface
  • Redis Commander - Redis key-value browser
Development Tools
Complete development environment accessible through web browsers for coding and testing.
  • VS Code Server - Full IDE in browser
  • MCP Inspector - Model Context Protocol testing
  • Jupyter Lab - Data science notebooks
Automation & Infrastructure
Container management and workflow automation for DevOps and business process automation.
  • Portainer - Docker container management UI

Network Architecture

All 16 containers operate on a single Docker bridge network (v2_network) for simplified communication and management. Services discover each other using container names, eliminating complex networking configurations.
